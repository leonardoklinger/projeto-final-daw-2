# Sistema de Leilão
Projeto final sobre a matéria desenvolvimento Web II IFG-Campus Inhumas Goiás.

# Requisitos
- PostgreSQL
- SpringBoot
- Java
- Visual Studio Code (Ou da sua preferência).
- IntelliJ  (Ou da sua preferência).
